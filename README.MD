#Design of Digital Systems for Radiation Detection.

Questo archivio raccoglie il codice sorgente, i testbench e i file di configurazione sviluppati per le esercitazioni pratiche e le relazioni del corso di Design of Digital Systems.

La repository è organizzata in cartelle dedicate per ciascuna attività di laboratorio, contenenti i file necessari per le relazioni nelle cartelle `Consegne` e un backup dei file sorgente utilizzati per le varie esperienze, nelle cartelle `sources`.
